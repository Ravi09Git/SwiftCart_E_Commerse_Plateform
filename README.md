# SwiftCart - E-Commerce Microservices Platform

## Overview
This is a full-stack e-commerce application built using **Spring Boot**, **Microservices architecture**, and **Vanilla JavaScript**.

## Tech Stack
- **Backend:** Java 17, Spring Boot, Spring Data JPA
- **Database:** H2 (In-Memory)
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
- **Tools:** Eclipse IDE, Postman, Maven

## Features
- Dynamic product listing from Backend API.
- Responsive Shopping Cart functionality.
- Microservices-ready architecture.
- RESTful API design.
